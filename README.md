# Business Sales Performance Analytics

## Project Overview
This project analyzes Superstore sales data to identify:
- Sales trends
- Top-performing products
- Regional performance
- Category profitability

## Tools Used
- Excel
- Data Analytics
- Business Visualization

## Key Insights
- Technology category generated the highest sales.
- West region showed strong performance.
- Certain products contributed significantly to revenue.
- Seasonal trends affected overall sales.

## Files Included
- Business_Sales_Analytics_Report.docx
- Cleaned_Superstore_Data.xlsx
- README.md
## Outcome
The project provides business insights and recommendations that can help improve sales strategy and decision-making.
